# Vehicle Number Plate Detection

This Python project detects vehicle number plates from images using the PlateRecognizer API.

## Technologies Used
- Python
- Requests Library
- PlateRecognizer API

## How to Run
1. Add your API key in the code
2. Place the vehicle image in the project folder
3. Run the Python script
4. The detected vehicle number plate will be printed
